# Instruções para o Agente

Este projeto é desenvolvido com apoio de agentes de IA.

## Diretrizes gerais

- Gere código simples, funcional e legível
- Evite complexidade desnecessária
- Utilize TypeScript no frontend e backend
- Mantenha boa organização e modularização
- Não invente funcionalidades não solicitadas
- Evite alucinações (não criar código fictício)

## Estrutura esperada

- Frontend: React com TypeScript
- Backend: Node.js com TypeScript
- Separação clara entre frontend e backend

## Qualidade do código

- Priorize clareza sobre otimização prematura
- Evite duplicação de código
- Escreva funções pequenas e bem definidas

## Execução

- Sempre explique como rodar o projeto
- Inclua instruções de instalação e execução

## Importante

- Pense antes de gerar código
- Caso não tenha certeza, peça esclarecimentos
